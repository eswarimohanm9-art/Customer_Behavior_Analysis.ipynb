# Customer Behavior Analysis

## Overview

Analyzed customer purchasing behavior using an e-commerce dataset to identify spending patterns, payment preferences, and customer segments.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Features

* Data Cleaning
* Data Visualization
* Customer Spending Analysis
* Payment Method Analysis
* Customer Segmentation
